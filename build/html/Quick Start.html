Quick Start¶

Installation¶

Currently, there are only two methods to install tigersec, PyPI and building from source. However, soon, tigersec may be added to the Debian Package Registry and the Ubuntu Package Registry

Installing from PyPI¶

  1. Run pip install tigersec

  2. Verify that tigersec works

Installing from Source¶

Version 1¶

  1. Download tigersec from the source as a ZIP archive

  2. Unzip tigersec

  3. Run sudo python setup.py install

  4. Verify that tigersec works

Version 2¶

  1. Run git clone https://github.com/dssecret/tigersec.git

  2. Then run sudo python setup.py install

  3. Verify that tigersec works

Version 3¶

  1. Run pip install git+git://github.com/dssecret/tigersec.git or run pip install git+https://github.com/dssecret/tigersec.git

  2. Verify that tigersec works
